# Deducto (AI Tax Manager)

Deducto is an Android-first expense & tax tracking app for freelancers and solo entrepreneurs in India. The repo contains the product direction, design assets, and the Android app skeleton to begin development.

What you'll find in this repo
- `prd_ai_expense_tax_tracker_for_freelancers.md` — Product Requirements (updated for Deducto / India)
- `android-app/` — Android module skeleton (Kotlin + Jetpack Compose)
- `design/` — UI reference images and logos

Getting started
1. Clone the repo
2. Open `android-app` in Android Studio
3. Run `./gradlew assembleDebug` to build
